Usage: ​

js
import yahooFinance from 'yahoo-finance2';
const queryOptions = { lang: 'en-US', formatted: false, region: 'US' };
const result = await yahooFinance.options('AYX', queryOptions);

API ​

js
await yahooFinance.options(query, queryOptions, moduleOptions);

Symbol ​

Symbol name as used by Yahoo (often the stock ticker). You can find it using autoc or search. You can also provide an array of symbols, and you'll receive an array of results back.

Query Options ​

NameTypeDefaultDescription
formattednumber5The max amount of symbols that can be returned.
langstring"en-US"
regionstring"US"
dateDate*undefinedThe expiration date of the options chain to be fetched.

Date* can be:

  • A Date instance, e.g. new Date(something)
  • A string that can be parsed by Date(), e.g. "2020-01-01".
  • A number representing unix timestamp WITHOUT milliseconds (as Yahoo expects)
